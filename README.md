# Commands
`Prefix "cc"`

* Logs - Shows logs.
* ClearLogs - Clears logs in console and in the log file.
* Ban - Bans specified guild member.
* Kick - Kicks specified guild member.
* Servers - Shows what server the bot is in.
* Hello - Responds to the user with hello.
